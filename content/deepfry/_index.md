## Commands
- **[deepfry](#deepfry)** Deepfries images.
- **[nuke](#nuke)** Demolishes images.

## deepfry
```
Syntax: gary deepfry [link]
Alias: gary df
```
**Deepfries images.**

The optional parameter "link" can be either a member or a direct link to an image.
## nuke
```
Syntax: gary nuke [link]
```
**Demolishes images.**

The optional parameter "link" can be either a member or a direct link to an image.
